

## Description

Version: 1.4

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
Format: yarn start <x>,<y>,<NORTH|SOUTH|EAST|WEST> <[FBLR]+>
$ yarn start "1,2,NORTH" FFFFRFF
$ yarn start "-1,-4,EAST" RFFLLB
```

## Running the tests
```bash
# unit tests
$ yarn test
```
